# GitHub Projects

## GitHub Projects とは

GitHub Projects は、GitHub が提供するプロジェクト管理ツールです。Issue や Pull Request をボード形式で管理し、タスクの進捗状況を可視化できます。

## Dotto Backlog

Dotto では、[Dotto Backlog](https://github.com/orgs/fun-dotto/projects/1/views/1) という GitHub Project を使用して、開発タスクを管理しています。

各タスクは Issue として作成され、以下のような状態で管理されます：

- **未対応**: まだ着手していないタスク
- **対応中**: 現在作業中のタスク
- **レビュー中**: Pull Request が作成され、レビュー待ちのタスク
- **完了**: マージされ、完了したタスク

タスクを選ぶ際は、この Backlog を参照して、適切なタスクを選択してください。

## タスクの選び方

基本的に、`1. 優先度の高い順`、`2. 規模の小さい順`、`3. 作成日順`に並んでいます。

そのため、特にこだわりがなければ、上から順番にタスクを消化していくと良いでしょう。

選んだタスクの `Assignees` を自分に設定、Status を `対応中` に変更します。
